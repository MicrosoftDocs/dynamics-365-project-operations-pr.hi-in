<span data-ttu-id="d2217-119">किसी/एक उद्धरण पंक्ति के लिए लेन-देन श्रेणियों पर प्रभार्य को परिभाषित किया जाता है, और यह केवल **खर्च** लेन-देन वर्ग पर लागू होता है.</span><span class="sxs-lookup"><span data-stu-id="d2217-119">Chargeability is defined on transaction categories for a  quote line and only applies to the **Expense** transaction class.</span></span> <span data-ttu-id="d2217-120">यदि किसी/एक परियोजना उद्धरण पंक्ति पर **खर्च को शामिल करें** क्षेत्र को **नहीं** पर सेट किया जाता है, तो **प्रभार्य श्रेणियां** टैब उपलब्ध नहीं होगा.</span><span class="sxs-lookup"><span data-stu-id="d2217-120">If the field, **Include Expenses** is set to **No** on a project quote line, the **Chargeable Categories** tab isn't available.</span></span>
