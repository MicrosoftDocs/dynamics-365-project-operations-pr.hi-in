# <a name="whats-new-may-2021---project-operations-lite-deployment"></a>क्या नया है मई 2021 - Project Operations लाइट परिनियोजन

_इस पर लागू होता है: लाइट परिनियोजन - प्रोफ़ॉर्मा इनवॉइस करने के लिए डील_

यह आलेख निम्न Dynamics 365 Project Operations घटक और संस्करणों पर लागू होता है:

   - Dataverse पर Project Operations परिवेश संस्करण 4.10.0.186.

## <a name="features-included-in-this-release"></a>इस रिलीज़ में शामिल की गई सुविधाएँ

इस रिलीज़ में निम्न सुविधाएँ शामिल की गई हैं:

- [शेड्यूलिंग मोड](../../project-management/scheduling-modes.md): परियोजना प्रबंधक अब निश्चित कर सकते हैं कि क्या किसी परियोजना को निश्चित अवधि, निश्चित कार्य या निश्चित इकाइयों के शेड्यूलिंग मोड का उपयोग करके प्रबंधित किया जाना चाहिए.
