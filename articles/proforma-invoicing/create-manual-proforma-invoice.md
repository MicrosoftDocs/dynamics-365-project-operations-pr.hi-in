<span data-ttu-id="f80ea-162">जब आप प्रोजेक्ट के बीजक का ड्राफ्ट तैयार करते हैं तो समय, खर्च और सामग्री उपयोग की प्रविष्टियों को अनुमोदन प्रदान किए जाने पर बिक्री के बिल न किए गए सभी लेनदेन बीजक में लाए जाते हैं.</span><span class="sxs-lookup"><span data-stu-id="f80ea-162">When you create a draft project invoice, all unbilled sales transactions that were created when the time, expense, and material usage entries were approved are pulled onto the invoice.</span></span> <span data-ttu-id="f80ea-163">बीजक के मसौदा अवस्था में होने पर आप निम्नलिखित एडजस्ट कर सकते हैं:</span><span class="sxs-lookup"><span data-stu-id="f80ea-163">You can make the following adjustments while the invoice is still in a draft stage:</span></span>

- <span data-ttu-id="f80ea-164">बीजक लाइन के विवरण को हटाना या एडिट करना।</span><span class="sxs-lookup"><span data-stu-id="f80ea-164">Delete or edit invoice line details.</span></span>
- <span data-ttu-id="f80ea-165">बिल की प्रमात्रा और प्रकार को एडिट करना या एडजस्ट करना।</span><span class="sxs-lookup"><span data-stu-id="f80ea-165">Edit and adjust the quantity and billing type.</span></span>

<span data-ttu-id="f80ea-166">बीजक को संपुष्ट करने के लिए **संपुष्ट** चुने।</span><span class="sxs-lookup"><span data-stu-id="f80ea-166">Select **Confirm** to confirm an invoice.</span></span> <span data-ttu-id="f80ea-167">संपुष्ट की कार्रवाई एकतरफा कार्रवाई होती है।</span><span class="sxs-lookup"><span data-stu-id="f80ea-167">The Confirm action is a one-way action.</span></span> <span data-ttu-id="f80ea-168">जब आप **संपुष्ट** चुनते हैं तो सिस्टम बीजक को रीड-ओनली बना देता है और प्रत्येक बीजक लाइन के लिए प्रत्येक बीजक लाइन से बिल की गई वास्तविक बिक्री तैयार की जाती है।</span><span class="sxs-lookup"><span data-stu-id="f80ea-168">When you select **Confirm**, the system makes the invoice read-only and creates billed sales actuals from each invoice line detail for each invoice line.</span></span> <span data-ttu-id="f80ea-169">यदि बीजक लाइन के विवरण बिन न की गई वास्तविक बिक्री दर्शाती है तो सिस्टम वास्तविक रूप से बिल न की गई बिक्री को वापस भी कर देता है।</span><span class="sxs-lookup"><span data-stu-id="f80ea-169">If the invoice line detail references an unbilled sales actual, the system also reverses the unbilled sales actual.</span></span> <span data-ttu-id="f80ea-170">(बीजक लाइन का कोई विवरण जो समय और खर्च की प्रविष्टि से बनाया गया है, बिल न की गई वास्तविक बिक्री को दर्शाता है।) जनरल लेजर इंटीग्रेशन सिस्टम लेखांकन के प्रयोजनार्थ प्रोजेक्ट कार्य को वापस करने के लिए इसका इस्तेमाल कर सकता है।</span><span class="sxs-lookup"><span data-stu-id="f80ea-170">(Any invoice line detail that was created from a time or expense entry will reference an unbilled sales actual.) ### <a name="correct-a-confirmed-invoice"></a><span data-ttu-id="f80ea-171">एक पुष्टि किए गए इनवॉइस को ठीक करें</span><span class="sxs-lookup"><span data-stu-id="f80ea-171">Correct a confirmed invoice</span></span>

<span data-ttu-id="f80ea-172">पुष्टि की गई इनवॉइस संपादित की जा सकती हैं (सही किए गए).</span><span class="sxs-lookup"><span data-stu-id="f80ea-172">Confirmed invoices can be edited (corrected).</span></span> <span data-ttu-id="f80ea-173">जब आप एक संपुष्ट बीजक में सुधार करते हैं तो एक सुधारित बीजक का एक नया मसौदा तैयार होता है।</span><span class="sxs-lookup"><span data-stu-id="f80ea-173">When you correct a confirmed invoice, a new draft corrective invoice is created.</span></span> <span data-ttu-id="f80ea-174">क्योंकि मान्यता यह होती है कि आप मूल बीजक में किए गए सभी लेनदेनों और मात्राओं को वापस करना चाहते हैं जिसमें मूल बीजक के सभी लेनदेन शामिल होते हैं और उसकी सभी मात्राएँ 0 (शून्य) हो जाती हैं।</span><span class="sxs-lookup"><span data-stu-id="f80ea-174">Because the assumption is that you want to reverse all the transactions and quantities from the original invoice, this corrective invoice includes all the transactions from the original invoice, and all the quantities on it are 0 (zero).</span></span>

<span data-ttu-id="f80ea-175">यदि किसी लेनदेन में किसी सुधार की ज़रूरत नहीं होती है तो आप उन्हें सुधार किए गए बीजक के मसौदे से हटा सकते हैं।</span><span class="sxs-lookup"><span data-stu-id="f80ea-175">If any transactions don't require correction, you can remove them from the draft corrective invoice.</span></span> <span data-ttu-id="f80ea-176">यदि आप किसी आंशिक मात्रा को ही बदलना या वापस करना चाहते हैं तो आप लाइन विवरण में **क्वान्टिटी** फील्ड को एडिट कर सकते हैं।</span><span class="sxs-lookup"><span data-stu-id="f80ea-176">If you want to reverse or return only a partial quantity, you can edit the **Quantity** field on the line detail.</span></span> <span data-ttu-id="f80ea-177">यदि आप बीजक लाइन विवरण को खोलते हैं, तो आप मूल बीजक की मात्रा को देख सकते हैं।</span><span class="sxs-lookup"><span data-stu-id="f80ea-177">If you open the invoice line detail, you can see the original invoice quantity.</span></span> <span data-ttu-id="f80ea-178">इसके बाद आप चालू बीजक मात्रा को एडिट कर सकते हैं ताकि यह मूल बीजक की मात्रा से कम या अधिक हो।</span><span class="sxs-lookup"><span data-stu-id="f80ea-178">You can then edit the current invoice quantity so that it's less than or more than the original invoice quantity.</span></span>

<span data-ttu-id="f80ea-179">जब आप सुधारित बीजक को संपुष्ट करते हैं तो मूल रूप से बिल की गई वास्तविक बिक्री उलट जाती है और नई बिल की गई वास्तविक बिक्री तैयार की जाती है।</span><span class="sxs-lookup"><span data-stu-id="f80ea-179">When you confirm a corrective invoice, the original billed sales actual is reversed, and a new billed sales actual is created.</span></span> <span data-ttu-id="f80ea-180">यदि मात्रा को कम किया जाता है तो अंतर के कारण नई बिल न की गई वास्तविक बिक्री भी तैयार कर ली जाती है।</span><span class="sxs-lookup"><span data-stu-id="f80ea-180">If the quantity was reduced, the difference will cause a new unbilled sales actual to be created too.</span></span> <span data-ttu-id="f80ea-181">उदाहरण के लिए, यदि मूल बिक्री का बिल आठ घंटे के लिए था, और सुधार करने योग्य इनवॉइस लाइन विवरण में छह घंटे की कम हुई मात्रा है, तो मूल बिल बिक्री लाइन पूर्णतया बदल जाती है और दो नए वास्तविक आंकड़े बनाए जाते हैं:</span><span class="sxs-lookup"><span data-stu-id="f80ea-181">For example, if the original billed sale was for eight hours, and the corrective invoice line detail has a reduced quantity of six hours, the original billed sales line is reversed and two new actuals are created:</span></span>
