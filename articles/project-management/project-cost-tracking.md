<span data-ttu-id="30cf8-118">प्रत्येक कार्य कार्य पर लागत विचरण का अनुमान दिखाता है.</span><span class="sxs-lookup"><span data-stu-id="30cf8-118">Each task shows a projection of the cost variance on the task.</span></span> <span data-ttu-id="30cf8-119">यदि पूरा होने पर लागत का अनुमान नियोजित लागत से अधिक है, तो कार्य बजट से अधिक होने का अनुमान है.</span><span class="sxs-lookup"><span data-stu-id="30cf8-119">If the cost estimate at complete is more than the planned cost, the task is projected to go over budget.</span></span> <span data-ttu-id="30cf8-120">यदि पूरा होने पर लागत का अनुमान नियोजित लागत से कम है, तो कार्य बजट के अंदर समाप्त होने का अनुमान है.</span><span class="sxs-lookup"><span data-stu-id="30cf8-120">If the cost estimate at complete is less than the planned cost, the task is projected to finish under budget.</span></span>

## <a name="reprojecting-costs-on-leaf-node-tasks"></a><span data-ttu-id="30cf8-133">लीफ़ नोड कार्यों पर लागत को फिर से अनुमानित करना</span><span class="sxs-lookup"><span data-stu-id="30cf8-133">Reprojecting costs on leaf node tasks</span></span>

4. <span data-ttu-id="30cf8-146">समरी टास्क के पूर्ण मूल्य पर लागत का रूट नोड तक दोबारा परिकलन किया जाता है.</span><span class="sxs-lookup"><span data-stu-id="30cf8-146">The cost at complete value of the summary tasks all the way to the root node are recalculated.</span></span>

## <a name="reprojecting-costs-on-summary-tasks"></a><span data-ttu-id="30cf8-147">संक्षेपित कार्यों पर लागत का फिर से पेश करना</span><span class="sxs-lookup"><span data-stu-id="30cf8-147">Reprojecting costs on summary tasks</span></span>

<span data-ttu-id="30cf8-148">आप संक्षेपित कार्यों या कंटेनर कार्यों पर श्रम लागत को फिर से पेश कर सकते हैं.</span><span class="sxs-lookup"><span data-stu-id="30cf8-148">You can reproject labor costs on summary tasks or container tasks.</span></span> <span data-ttu-id="30cf8-149">हालांकि, आप **परियोजना** पेज पर **ट्रैकिंग** टैब पर एक संक्षेपित परियोजना कार्य पर श्रम लागत को फिर से सीधे पेश नहीं कर सकते हैं.</span><span class="sxs-lookup"><span data-stu-id="30cf8-149">However, you can't directly reproject labor cost on a summary project task on the **Tracking** tab on the **Project** page.</span></span> <span data-ttu-id="30cf8-150">लीफ़ नोड कार्यों के समान, संक्षेपित और कंटेनर कार्यों को **प्रयास की ट्रैकिंग** दृश्य का उपयोग करके फिर से पेश किया जा सकता है.</span><span class="sxs-lookup"><span data-stu-id="30cf8-150">Similar to leaf node tasks, reprojecting summary and container tasks can be done using the **Effort Tracking** view.</span></span> <span data-ttu-id="30cf8-151">इस दृश्य में, आप सारांश कार्य पर शेष प्रयास का फिर अनुमान लगा सकते हैं जिससे सारांश कार्य पर शेष लागत की पुनर्गणना हो सकती है.</span><span class="sxs-lookup"><span data-stu-id="30cf8-151">In this view, you can reproject the remaining effort on a summary task causing a recalculation of the remaining cost on the summary task.</span></span> <span data-ttu-id="30cf8-152">इसके काम करने का वर्णन निम्नलिखित है.</span><span class="sxs-lookup"><span data-stu-id="30cf8-152">The following is a description of how this works.</span></span>

1. <span data-ttu-id="30cf8-153">एक प्रोज़ेक्ट प्रबंधक कार्य पर एक नए अनुमान के साथ शेष प्रयास के डिफ़ॉल्ट मान को अपडेट करके सारांश कार्यों पर प्रयास पर फिर से अनुमान लगा सकता है.</span><span class="sxs-lookup"><span data-stu-id="30cf8-153">A project manager can reproject effort on summary tasks by updating the default value of the remaining effort with a new estimate on the task.</span></span> <span data-ttu-id="30cf8-154">यह अपडेट सारांश कार्य के पूरा होने पर अनुमान (EAC), प्रगति प्रतिशत, और कार्य के अनुमानित प्रयास के विचरण पर अनुमान की पुनर्गणना करता है.</span><span class="sxs-lookup"><span data-stu-id="30cf8-154">This update causes a recalculation of the summary task's estimate at complete (EAC), progress percentage, and the projected effort variance on a task.</span></span> <span data-ttu-id="30cf8-155">EAC, ETC और सारांश कार्यों पर प्रगति प्रतिशत की भी पुनर्गणना होती है और प्रयास विचरण के एक नए प्रक्षेपण का उत्पादन करता है.</span><span class="sxs-lookup"><span data-stu-id="30cf8-155">The EAC, ETC, and progress percentage on the summary tasks are also recalculated and produce a new projection of effort variance.</span></span>
2. <span data-ttu-id="30cf8-156">सारांश कार्य पर शेष प्रयास के लिए नए मूल्य के आधार पर, सिस्टम **लागत ट्रैकिंग** व्यू पर शेष लागत की गणना करता है.</span><span class="sxs-lookup"><span data-stu-id="30cf8-156">Based on the new value for remaining effort on the summary task, the system calculates the remaining cost on the **Cost Tracking** view.</span></span> <span data-ttu-id="30cf8-157">शेष प्रयास के आधार पर शेष लागत की गणना करने के लिए, सिस्टम पहले नियोजित लागत या नियोजित प्रयास के रूप में सारांश कार्य पर एक घंटे के प्रयास की औसत लागत की गणना करता है.</span><span class="sxs-lookup"><span data-stu-id="30cf8-157">To calculate the remaining cost based on the remaining effort, the system first calculates the average cost of one hour of effort on the summary task as planned cost or planned effort.</span></span> <span data-ttu-id="30cf8-158">प्रति घंटे औसत लागत का उपयोग सारांश कार्य पर नए अनुमानित शेष प्रयास की लागत की गणना करने के लिए किया जाता है.</span><span class="sxs-lookup"><span data-stu-id="30cf8-158">The average cost per hour is used to calculate the cost of the newly projected remaining effort on the summary task.</span></span>
3. <span data-ttu-id="30cf8-159">सारांश कार्य पर पूर्ण और लागत खपत प्रतिशत पर लागत की फिर से गणना की जाती है.</span><span class="sxs-lookup"><span data-stu-id="30cf8-159">The cost at complete and cost consumption percentage on the summary task are re-calculated.</span></span>
4. <span data-ttu-id="30cf8-160">नए पूर्ण मूल्य पर लागत को छोटे टास्क में उसी अनुपात में नीचे वितरित किया जाता है जिस अनुपात में टास्क में मूल योजनाबद्ध लागत था.</span><span class="sxs-lookup"><span data-stu-id="30cf8-160">The new cost at complete is distributed down to the child tasks in the same proportion as the original planned cost was on the task.</span></span>
5. <span data-ttu-id="30cf8-161">व्यक्तिगत कार्यों में से प्रत्येक के लिए लीफ़ नोड कार्यों तक के लिए पूर्ण मूल्य पर नई लागत की गणना की जाती है.</span><span class="sxs-lookup"><span data-stu-id="30cf8-161">The new cost at complete value on each of the individual tasks down to the leaf node tasks is calculated.</span></span> <span data-ttu-id="30cf8-162">इस मूल्य के आधार पर, नीचे लीफ़ नोड्स तक प्रभावित चाइल्ड कार्यों के लिए उनकी शेष लागत और लागत खपत प्रतिशत के पूर्ण मूल्य पर लागत के आधार पर पुनर्गणना की जाएगी.</span><span class="sxs-lookup"><span data-stu-id="30cf8-162">Based on this value, the affected child tasks down to the leaf nodes will have their remaining cost and cost consumption percentage recalculated based on the cost at complete value.</span></span> <span data-ttu-id="30cf8-163">इस मान के कारण टास्क की लागत भिन्नता का नया अनुमान करना होता है.</span><span class="sxs-lookup"><span data-stu-id="30cf8-163">This value results in a new projection for the cost variance of the task.</span></span> 


<span data-ttu-id="30cf8-164">**लागत प्रदर्शन** फ़ील्ड ट्रैकिंग डेटा से सेट किया जा सकता है.</span><span class="sxs-lookup"><span data-stu-id="30cf8-164">The **Cost performance** field can be set from the tracking data.</span></span> <span data-ttu-id="30cf8-165">जब **लागत ट्रैकिंग** दृश्य में रूट नोड के लिए लागत विचरण नकारात्मक है, तो आप इस फ़ील्ड को **अंडर बजट** सेट कर सकते हैं.</span><span class="sxs-lookup"><span data-stu-id="30cf8-165">When the cost variance for the root node in the **Cost tracking** view is negative, you can set this field to **Under Budget**.</span></span> <span data-ttu-id="30cf8-166">जब रूट नोड के लिए लागत विचरण धनात्मक होता है, तो आप मूल्य को **ओवर बजट** सेट कर सकते हैं.</span><span class="sxs-lookup"><span data-stu-id="30cf8-166">When the cost variance for the root node is positive, you can set the value to **Over Budget**.</span></span>
