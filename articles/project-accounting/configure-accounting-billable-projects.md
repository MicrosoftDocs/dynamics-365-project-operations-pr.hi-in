# <a name="configure-accounting-for-billable-projects"></a><span data-ttu-id="df1bb-103">बिल करने योग्य परियोजनाओं के लिए लेखांकन कॉन्फ़िगर करें</span><span class="sxs-lookup"><span data-stu-id="df1bb-103">Configure accounting for billable projects</span></span>

<span data-ttu-id="df1bb-104">_**पर लागू होता है:** साधन / गैर-स्टॉक आधारित परिदृश्यों के लिए Project Operations, Lite परिनियोजन - प्रोफार्मा इनवॉइसिंग के लिए डील_</span><span class="sxs-lookup"><span data-stu-id="df1bb-104">_**Applies To:** Project Operations for resource/non-stocked based scenarios, Lite deployment - deal to proforma invoicing_</span></span>

<span data-ttu-id="df1bb-105">Dynamics 365 Project Operations बिल बनाने योग्य परियोजनाओं के लिए विभिन्न लेखांकन विकल्पों का समर्थन करता है जिसमें समय और सामग्री और निश्चित मान के लेनदेन शामिल हैं.</span><span class="sxs-lookup"><span data-stu-id="df1bb-105">Dynamics 365 Project Operations supports various accounting options for billable projects that include time and material and fixed price transactions.</span></span>

- <span data-ttu-id="df1bb-106">**समय और सामग्री लेनदेन** : इन लेनदेन की इनवॉइस परियोजना में उपभोग के घंटे, व्यय, वस्तुओं, या शुल्क के उपभोग के आधार पर काम की प्रगति होने पर बनायी जाती हैं.</span><span class="sxs-lookup"><span data-stu-id="df1bb-106">**Time and material transactions** : These transactions are invoiced as the work progresses based on the consumption of hours, expenses, items, or fees on the project.</span></span> <span data-ttu-id="df1bb-107">इन लेनदेन की लागत का मिलान प्रत्येक लेनदेन की आमदनी से किया जा सकता है और काम की प्रगति होने पर परियोजना का इनवॉयस बनाया जाता है.</span><span class="sxs-lookup"><span data-stu-id="df1bb-107">These transaction costs can be matched with the revenue on each transaction and the project is invoiced as work progresses.</span></span> <span data-ttu-id="df1bb-108">लेनदेन होने पर परियोजना आमदनी भी प्राप्त की जा सकती है.</span><span class="sxs-lookup"><span data-stu-id="df1bb-108">Project revenue can be also accrued at the time when the transaction occurs.</span></span> <span data-ttu-id="df1bb-109">इनवॉयस के दौरान, आमदनी की पहचान की जाती है और अगर लागू हो तो इस राशि को रिवर्स कर दिया जाता है.</span><span class="sxs-lookup"><span data-stu-id="df1bb-109">During invoicing, revenue is recognized and if applicable, accrued revenue is reversed.</span></span>
- <span data-ttu-id="df1bb-110">**निर्धारित-कीमत लेनदेन** : इन लेनदेन का इनवॉयस बिलिंग शेड्यूल से बनाए जाते हैं, जो परियोजना अनुबंध पर आधारित होता है.</span><span class="sxs-lookup"><span data-stu-id="df1bb-110">**Fixed-price transactions** : These transactions are invoiced according to a billing schedule that is based on the project contract.</span></span> <span data-ttu-id="df1bb-111">निश्चित कीमत के लेनदेन के लिए आय को **पूर्ण अनुबंध** या **पूर्ण प्रतिशत** विधि के अनुसार समय समय पर इनवॉइस बनायी जा सकती है या गणना की जा सकती है और दर्ज किया जा सकती है.</span><span class="sxs-lookup"><span data-stu-id="df1bb-111">Revenue for fixed price transactions can be recognized at invoicing or calculated and posted periodically, according to the **Completed contract** or **Completed percentage** methods.</span></span>

<span data-ttu-id="df1bb-112">परियोजना को बिल के योग्य माना जाता है जब यह एक या अधिक अनुबंध लाइन से जुड़ी हो.</span><span class="sxs-lookup"><span data-stu-id="df1bb-112">A project is considered billable when it is associated with one or more contract lines.</span></span> <span data-ttu-id="df1bb-113">परियोजना अनुबंध लाइन अपने लिए स्पष्ट करती है कि बिलिंग की किस विधि और लेनदेन के किस प्रकार को अनुमति है.</span><span class="sxs-lookup"><span data-stu-id="df1bb-113">A project contract line defines for itself which billing method and transaction types are allowed.</span></span>

<span data-ttu-id="df1bb-114">उदाहरण के लिए, Fabrikam Robotics ने उपकरण अनुकूलन के लिए Adatum कॉर्पोरेशन का परियोजना अनुबंध हासिल किया.</span><span class="sxs-lookup"><span data-stu-id="df1bb-114">As an example, Fabrikam Robotics has won a project contract with Adatum corporation for equipment optimization.</span></span> <span data-ttu-id="df1bb-115">Adatum परियोजना खर्चों को कवर करने के लिए $10.000 USD की निश्चित धनराशि का भुगतान करेगा.</span><span class="sxs-lookup"><span data-stu-id="df1bb-115">Adatum will pay a fixed amount of $10.000 USD to cover incurred project expenses.</span></span> <span data-ttu-id="df1bb-116">यह निर्धारित कीमत बिलिंग विधि है.</span><span class="sxs-lookup"><span data-stu-id="df1bb-116">This is a fixed price billing method.</span></span> <span data-ttu-id="df1bb-117">परियोजना समय और शुल्क का बिल प्रति उपयोग के हिसाब से बनाया जाएगा.</span><span class="sxs-lookup"><span data-stu-id="df1bb-117">Project time and fees will be billed per use.</span></span> <span data-ttu-id="df1bb-118">यह समय और सामग्री बिलिगं विधि है.</span><span class="sxs-lookup"><span data-stu-id="df1bb-118">This is a time and material billing method.</span></span> <span data-ttu-id="df1bb-119">सभी कार्यों पर एकल परियोजना के तहत नज़र रखी जाएगी, जिसका नाम Adatum उपकरण अनुकूलन है.</span><span class="sxs-lookup"><span data-stu-id="df1bb-119">All of the work will be tracked under a single project named, Adatum equipment optimization.</span></span>

<span data-ttu-id="df1bb-120">परियोजना टीम का सदस्य Adatum उपकरण अनुकूलन परियोजना में आठ घंटे काम करता है.</span><span class="sxs-lookup"><span data-stu-id="df1bb-120">A project team member submits eight hours of work on the Adatum equipment optimization project.</span></span> <span data-ttu-id="df1bb-121">जब परियोजना प्रबंधक इस समय को स्वीकृत करता है, तो सिस्टम वास्तविक लेनदेन, इनवॉयस और खाता बनाने के लिए समय और सामग्री बिलिंग विधि का इस्तेमाल करता है.</span><span class="sxs-lookup"><span data-stu-id="df1bb-121">When the project manager approves this time, the system uses the time and material billing method to create actuals transactions, an invoice, and an account.</span></span> <span data-ttu-id="df1bb-122">यह लेनदेन निर्धारित कीमत आय अनुमान गणना में शामिल नहीं होगा.</span><span class="sxs-lookup"><span data-stu-id="df1bb-122">This transaction will not be included in the fixed price revenue estimate calculation.</span></span>

<span data-ttu-id="df1bb-123">अन्य परियोजना टीम सदस्य Adatum उपकरण अनुकूलन परियोजना के लिए $2000.00 डॉलर के लिए यात्रा व्यय सबमिट करता है.</span><span class="sxs-lookup"><span data-stu-id="df1bb-123">Another project team member submits a travel expense for $2000.00 USD against the Adatum equipment optimization project.</span></span> <span data-ttu-id="df1bb-124">जब परियोजना प्रबंधक इस अनुरोध को स्वीकृत करता है तो सिस्टम इस परियोजना के लिए वास्तविक लेनदेन और खाता बनाने के लिए निश्चित कीमत बिलिंग विधि का इस्तेमाल करता है.</span><span class="sxs-lookup"><span data-stu-id="df1bb-124">When the project manager approves this submission, the system uses a fixed price billing method to create actuals transactions and an account for this project expense.</span></span> <span data-ttu-id="df1bb-125">ग्राहक का इनवॉयस इस लेनदेन के आधार पर नहीं बनाया जाएगा.</span><span class="sxs-lookup"><span data-stu-id="df1bb-125">The customer will not be invoiced based on this transaction.</span></span> <span data-ttu-id="df1bb-126">इसके बजाय, वे अलग-अलग कॉन्फ़िगर किए गए तयशुदा बिलिंग का इस्तेमाल करके इनवॉइस बनाएंगे.</span><span class="sxs-lookup"><span data-stu-id="df1bb-126">Instead, they will be invoiced by using separately configured billing milestones.</span></span> <span data-ttu-id="df1bb-127">यह व्यय लेनदेन, व्यय अनुमानों के साथ, निश्चित कीमत आय अनुमान गणना में शामिल किया जाएगा.</span><span class="sxs-lookup"><span data-stu-id="df1bb-127">This expense transaction, along with expense estimates, will be included in the fixed price revenue estimate calculation.</span></span>

<span data-ttu-id="df1bb-128">परियोजना लेनदेन के लिए लेखांकन सिद्धांतों को परियोजना लागत और आय प्रोफाइल में स्पष्ट किया गया है.</span><span class="sxs-lookup"><span data-stu-id="df1bb-128">Accounting principles for project transactions are defined in project cost and revenue profiles.</span></span> <span data-ttu-id="df1bb-129">प्रत्येक परियोजना लेनदेन के लिए, सिस्टम कॉन्फ़िगर किए गए परियोजना लागत और आय प्रोफाइल नियमों का इस्तेमाल करके उचित परियोजना लागत और आय प्रोफ़ाइल निर्धारित करता है.</span><span class="sxs-lookup"><span data-stu-id="df1bb-129">For every project transaction, the system determines the appropriate project cost and revenue profile by using the project cost and revenue profile rules that have been configured.</span></span>
