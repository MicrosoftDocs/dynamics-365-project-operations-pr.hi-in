<span data-ttu-id="d2859-105">परियोजना-आधारित कोट पंक्ति में घटक और प्रभार्य घटक शामिल हो सकते हैं.</span><span class="sxs-lookup"><span data-stu-id="d2859-105">A project-based quote line can have included components and chargeable components.</span></span>

   - <span data-ttu-id="d2859-110">गैर प्रभार्य</span><span class="sxs-lookup"><span data-stu-id="d2859-110">Non-chargeable</span></span>

<span data-ttu-id="d2859-111">प्रभार्य घटकों को भूमिकाओं, और लेनदेन श्रेणियों में परिभाषित किया जा सकता है.</span><span class="sxs-lookup"><span data-stu-id="d2859-111">Chargeable components can be defined on roles and transaction categories.</span></span>
