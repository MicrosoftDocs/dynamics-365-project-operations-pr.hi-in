## <a name="allocating-estimated-effort-based-on-the-task-mode"></a><span data-ttu-id="64f72-151">टास्क मोड के आधार पर आकलित प्रयास आवंटित करना</span><span class="sxs-lookup"><span data-stu-id="64f72-151">Allocating estimated effort based on the task mode</span></span>

<span data-ttu-id="64f72-152">समय-चरणबद्ध दृश्य में, आप चयनित टाइम स्केल में, प्रयास के घंटों को प्रति यूनिट समय अवधि में आवंटित करते हुए टास्क के लिए आकलित कुल प्रयास को वितरित करते हैं।</span><span class="sxs-lookup"><span data-stu-id="64f72-152">In the time-phased view, you distribute the total effort that is estimated for the task by allocating effort hours per unit time period in the selected time scale.</span></span> <span data-ttu-id="64f72-153">टास्क मोड यह तय करता है कि टास्क की अवधि के दौरान कितने प्रयास आबंटित किए जाने हैं।</span><span class="sxs-lookup"><span data-stu-id="64f72-153">The task mode determines how effort is allocated across the duration of the task.</span></span> <span data-ttu-id="64f72-154">दो प्रकार के आबंटन हैं **ईवन** और **कार्य घंटों पर आधारित**.</span><span class="sxs-lookup"><span data-stu-id="64f72-154">The two kinds of allocation are **Even** and **Work hours–based**.</span></span>

### <a name="work-hours-based-allocation"></a><span data-ttu-id="64f72-155">कार्य घंटों पर आधारित आबंटन</span><span class="sxs-lookup"><span data-stu-id="64f72-155">Work hours-based allocation</span></span>
 
<span data-ttu-id="64f72-156">ऑटो-शेड्यूलिंग टास्क मोड में, टास्क रिसोर्स के लिए दैनिक डिफाल्ट घंटे पूर्ण कार्य घंटा दर पर सेट किए जाते हैं।</span><span class="sxs-lookup"><span data-stu-id="64f72-156">In auto-scheduling task mode, the daily default hours for task resources are set at the full work hour rate.</span></span> <span data-ttu-id="64f72-157">ऐसा तब किया जाता है जब प्रयास को समय-चरणबद्ध दृश्य में टास्क की अवधि के दौरान विभाजित करते हुए आबंटित किया जाता है।</span><span class="sxs-lookup"><span data-stu-id="64f72-157">This behavior applies when effort is allocated by splitting it across the duration of the task in the time-phased view.</span></span> <span data-ttu-id="64f72-158">उदाहरण के लिए, यदि आपका अनुमान है कि टास्क को **दिन** टाइम स्केल में एक रिसोर्स द्वारा पूरा किया जाएगा तो प्रति दिन के लिए आबंटित प्रयास प्रोजेक्ट कैलेंडर में निर्धारित प्रति दिन कार्य घंटों से अधिक नहीं होगा।</span><span class="sxs-lookup"><span data-stu-id="64f72-158">For example, if you estimate that a task will be completed by one resource in the **Day** time scale, the effort that is allocated per day won't exceed the work hours per day that are defined in the project calendar.</span></span> <span data-ttu-id="64f72-159">इस प्रकार, प्रयास आवंटन से हमेशा यह सुनिश्चित किया जाता है कि संसाधन का आकलन पूरे दिन के इस्तेमाल के लिए किया जाता है।</span><span class="sxs-lookup"><span data-stu-id="64f72-159">Therefore, the effort allocation always makes sure that the resources are estimated to be used for the full day.</span></span>

### <a name="even-allocation"></a><span data-ttu-id="64f72-160">ईवन आबंटन</span><span class="sxs-lookup"><span data-stu-id="64f72-160">Even allocation</span></span>
