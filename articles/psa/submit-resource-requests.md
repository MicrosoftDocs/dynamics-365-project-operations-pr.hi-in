# <a name="submit-resource-requests-project-service"></a><span data-ttu-id="e37f7-103">संसाधन अनुरोध सबमिट करना (Project Service)</span><span class="sxs-lookup"><span data-stu-id="e37f7-103">Submit resource requests (Project Service)</span></span>

[!INCLUDE[cc-applies-to-psa-app-1x-2x](../includes/cc-applies-to-psa-app-1x-2x.md)]

<span data-ttu-id="e37f7-104">कोई परियोजना अनुबंध हासिल कर लेने के बाद, आपको अपनी परियोजना के लिए स्टाफ की आवश्यकता होगी.</span><span class="sxs-lookup"><span data-stu-id="e37f7-104">Once you’ve won a project contract, you’ll need to staff your project.</span></span> <span data-ttu-id="e37f7-105">आपकी कार्य विश्लेषण संरचना, परियोजना पूरी करने के लिए आवश्यक संसाधन भूमिकाएँ पहले ही स्थापित कर चुकी है, तो अब समय है आवश्यकताओं के अनुरूप संसाधनों के लिए अनुरोध करने का.</span><span class="sxs-lookup"><span data-stu-id="e37f7-105">Your work breakdown structure already established the resource roles you need to complete the project, so now it’s time to request the resources that match the requirements.</span></span>  
  
1.  <span data-ttu-id="e37f7-106">**Project Service > परियोजना** पर जाएँ.</span><span class="sxs-lookup"><span data-stu-id="e37f7-106">Go to **Project Service > Projects**.</span></span>  
  
2.  <span data-ttu-id="e37f7-107">जिस परियोजना पर आप काम करना चाहते हैं, उस पर क्लिक करें.</span><span class="sxs-lookup"><span data-stu-id="e37f7-107">Click the project you want to work on.</span></span>  
  
3.  <span data-ttu-id="e37f7-108">स्क्रीन के ऊपर दी गई पट्टी में, परियोजना नाम के आगे स्थित नीचे तीर का चयन करें, और उसके बाद **संसाधन आवश्यकताएँ** पर क्लिक करें.</span><span class="sxs-lookup"><span data-stu-id="e37f7-108">In the bar across the top of the screen, select the down arrow next to the project name, and then click **Resource Requirements**.</span></span>  
  
4.  <span data-ttu-id="e37f7-109">संसाधन अनुरोध सबमिट करने के लिए, उन भूमिकाओं को चुनें, जिनके लिए आप संसाधनों का अनुरोध करना चाहते हैं और उसके बाद **अनुरोध सबमिट करें** पर क्लिक करें.</span><span class="sxs-lookup"><span data-stu-id="e37f7-109">To submit resource requests, select the roles you want to request resources for, and then click **Submit Request**.</span></span>  
  
5.  <span data-ttu-id="e37f7-110">इसके बजाए, संसाधन को हार्ड बुक करने के लिए, भूमिका चुनें और उसके बाद **हार्ड बुक** पर क्लिक करें.</span><span class="sxs-lookup"><span data-stu-id="e37f7-110">To hard book a resource instead, select the role, and then click **Hard Book**.</span></span> <span data-ttu-id="e37f7-111">यह सभी संसाधनों और उस रंग कोडिंग के साथ उनकी उपलब्धता का कैलेंडर दृश्य दिखाता है, जो संसाधन की बुकिंग स्थिति का विज़ुअल दृश्य प्रदान करती है.</span><span class="sxs-lookup"><span data-stu-id="e37f7-111">This shows a calendar view of all resources and their availability, with color coding that provides a visual view of a resource’s booking status.</span></span> <span data-ttu-id="e37f7-112">अपनी खोज को अनुकूलित करने के लिए निम्न में से कोई कार्य करें:</span><span class="sxs-lookup"><span data-stu-id="e37f7-112">Do any of the following to optimize your search:</span></span>  
  
    -   <span data-ttu-id="e37f7-113">स्क्रीन के ऊपरी भाग से कोई दृश्य या सॉर्ट क्रम चुनें.</span><span class="sxs-lookup"><span data-stu-id="e37f7-113">Select a view or sort order from the top of the screen.</span></span> <span data-ttu-id="e37f7-114">आप चुन सकते हैं कि **दिन**, **सप्ताह** या **माह** के लिए उपलब्धता दिखानी है या नहीं और पिछली या अगली समय अवधि को देखने के लिए तीरों का उपयोग करें.</span><span class="sxs-lookup"><span data-stu-id="e37f7-114">You can also select whether to show availability for the **Day**, **Week**, or **Month**, and use the arrows to view the previous or next time period.</span></span> <span data-ttu-id="e37f7-115">किसी विशिष्ट संसाधन को ढूंढने के लिए खोज बॉक्स का उपयोग करें.</span><span class="sxs-lookup"><span data-stu-id="e37f7-115">Use the search box to find a specific resource.</span></span>  
  
    -   <span data-ttu-id="e37f7-116">प्रदर्शन को बदलने के लिए **विकल्प** पर क्लिक करें.</span><span class="sxs-lookup"><span data-stu-id="e37f7-116">Click **Options** to change what displays.</span></span> <span data-ttu-id="e37f7-117">उदाहरण के लिए, आप एक ऐसे लेजेंड को दिखा सकते हैं जो विभिन्न प्रकार की बुकिंग के लिए रंग योजना दिखाता है या आप केवल कुछ प्रकार की बुकिंग को दिखाना चुन सकते हैं (उदाहरण के लिए, हार्ड बुकिंग या सॉफ़्ट बुकिंग).</span><span class="sxs-lookup"><span data-stu-id="e37f7-117">For example, you can show a legend that shows the color scheme for the different types of bookings, or you can choose to show only a certain type of booking (for example, hard booking or soft booking).</span></span>  
  
    -   <span data-ttu-id="e37f7-118">**फ़िल्टर** फलक प्रदर्शित करने के लिए **फ़िल्टर** पर क्लिक करें, जो संगठनात्मक इकाई, भूमिका, कौशल या अन्य एट्रिब्यूट के अनुसार आपको अपने परिणामों को फ़िल्टर करने देता है.</span><span class="sxs-lookup"><span data-stu-id="e37f7-118">Click **Filter** to display the **Filter** pane, which lets you filter your results by organizational unit, role, skill, or other attributes.</span></span>  
  
    -   <span data-ttu-id="e37f7-119">जिस संसाधन को आप बुक करना चाहते हैं उसके लिए कैलेंडर में, संसाधन बुक करने की समय अवधि चुनें और उसके बाद या तो संसाधन बुक करने के लिए **बुक करें** पर क्लिक करें या बुकिंग का प्रस्ताव देने के लिए **बुकिंग का प्रस्ताव दें** पर क्लिक करें और अनुरोधकर्ता को अनुमोदन करने दें.</span><span class="sxs-lookup"><span data-stu-id="e37f7-119">In the calendar for the resource you want to book, select the time period to book the resource, and either click **Book** to book the resource, or **Propose Booking** to propose the booking and have the requestor approve it.</span></span>  
  
6.  <span data-ttu-id="e37f7-120">आपके संसाधन अनुरोधों की संसाधन प्रबंधक द्वारा समीक्षा और अनुमोदन किया जाना चाहिए.</span><span class="sxs-lookup"><span data-stu-id="e37f7-120">A resource manager needs to review and approve your resource requests.</span></span>
