# <a name="whats-new-march-2021---project-operations-for-resourcenon-stocked-based-scenarios"></a>नया क्या है मार्च 2021 - संसाधन/गैर-स्टॉक -आधारित परिदृश्यों के लिए Project Operations

_**इस पर लागू होता है:** संसाधन/गैर-स्टॉक -आधारित परिदृश्यों के लिए Project Operations_

यह लेख निम्नलिखित पर लागू होता है Dynamics 365 Project Operations घटक और संस्करण:

- Dataverse पर Project Operations परिवेश संस्करण 4.8.0.91 
- Dynamics 365 Finance पर्यावरण संस्करण 10.0.16 . पर परियोजना प्रबंधन और लेखांकन 

## <a name="quality-updates"></a>गुणवत्ता अद्यतन

| नियोजन और ट्रैकिंग | 2154306 | संसाधन-आधारित परिदृश्यों के लिए Project Operations में व्यय अनुमान हटाने से जुड़ी समस्याओं को ठीक किया गया है. |
