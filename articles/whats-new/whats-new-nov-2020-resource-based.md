# <a name="whats-new-november-2020---project-operations-for-resourcenon-stocked-based-scenarios"></a>नया क्या है नवंबर 2020 - संसाधन/गैर-स्टॉक -आधारित परिदृश्यों के लिए Project Operations

_**इस पर लागू होता है:** संसाधन/गैर-स्टॉक -आधारित परिदृश्यों के लिए Project Operations_

यह आलेख निम्न Dynamics 365 Project Operations घटक और संस्करणों पर लागू होता है:

- CDS पर Project Operations परिवेश संस्करण 4.4.0.70

### <a name="regulatory-updates"></a>नियामक अद्यतन
फाइनेंस और ऑपरेशन ऐप के लिए नियामक अद्यतनों के बारे में जानकारी के लिए, [नियामक अद्यतन](/dynamics365/finance/localizations/regulatory-updates) देखें. आप LCS में साइन इन भी कर सकते हैं और समस्या खोज उपकरण का उपयोग करके नियोजित नियामक अद्यतन देख सकते हैं. समस्या खोज के चलते आप देश, सुविधा के प्रकार और रिलीज़ के आधार पर खोज सकते हैं.
